# Code of Conduct

We want Agent Broker to be useful, technically serious, and easy to collaborate on.

## Expected behavior

- Be respectful and direct.
- Focus on technical substance and reproducible reports.
- Assume good faith, especially in review and issue discussion.
- Keep feedback specific, actionable, and grounded in the code or documented behavior.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Discriminatory or demeaning language.
- Deliberate bad-faith issue filing, trolling, or spam.
- Publishing private information without consent.

## Maintainer standards

Maintainers should:

- explain decisions clearly
- keep scope disciplined
- prefer reproducible bug reports over debate-by-assertion
- enforce this code of conduct consistently

## Reporting conduct issues

If you experience or witness unacceptable behavior, contact the maintainer privately instead of escalating in public threads.

For security-related concerns, use the guidance in [SECURITY.md](./SECURITY.md).
